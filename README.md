# RaspiInkyBusSchedule
Raspberry Pi based Bus Schedule using Pimoroni's Inky Impression eInk Display.

BUS_CODE = "" # https://www.openstreetmap.org/node/533791098#map=18/52.21592/0.10698&layers=TD
